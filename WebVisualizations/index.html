Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from OpenWeatherMap API to assemple a dataset on over 500 cities.


After assempling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at includedL temperature, cloudiness, wind speed, and hjumidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
